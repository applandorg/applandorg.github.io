AppMap is an open-source software analysis tool that generates visual depictions of running code for the purpose of understanding a codebase. AppMap records “Scenarios” from running code by observing its tests and doing analysis on the code itself.

Viewing an AppMap in VSCode looks like this…

The goal is for you to successfully install the AppLand extension for Visual Studio Code, generate AppMap data for one of your projects, and answer some questions that will help us improve the extension.

You will need two things:

  1. Visual Studio Code
  2. A Ruby project with RSpec, Minitest, or Cucumber test coverage. Preferably a Rails application with database queries and HTTP requests.

Step 1: Install The AppMap extension.

Search for AppMap in the extensions tab of Visual Studio Code, or get it from the Visual Studio Code Marketplace page.

Step 2: Install and configure the appmap-ruby gem to generate AppMap data

You can find the instructions on how to do that here.
